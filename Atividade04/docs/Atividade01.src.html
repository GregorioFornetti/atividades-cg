class Atividade01.src.ImageIO.ImageWriter(image: Union[ndarray, Image, Image])[código fonte]

Base: object

Construtor da classe ImageWriter. Recebe um objeto ou uma matriz representando uma imagem, para ser posteriormente salva ou visualizada.

—

Parâmetros:

  • image: Union[np.ndarray, Image, Image.Image] - Imagem a ser salva ou visualizada. Pode ser um objeto PIL.Image ou uma matriz numpy. A matriz numpy pode possuir valores entre 0 e 1 e ser do tipo float ou ser uma matriz do tipo uint8. Pode ser também um objeto Image, que é uma classe definida no arquivo Image.py.

Atividade01.src.tutorial module

Implementa o escritor de arquivos PPM, conforme descrito no tutorial: https://raytracing.github.io/books/RayTracingInOneWeekend.html?#outputanimage

O código está escrito originalmente em C++, mas foi adaptado para Python.

Para visualizar a imagem gerada, utilize o site: https://www.cs.rhodes.edu/welshc/COMP141_F16/ppmReader.html
